# 🏡 EstateHub – Modern Real Estate Marketplace

EstateHub is a full-stack real estate web application built to transform the way people buy, sell, and rent properties online. Designed with a modern user interface and secure backend architecture, EstateHub delivers a seamless digital property browsing experience for users and real estate businesses.

Whether you are searching for your dream home, listing premium properties, or managing real estate data, EstateHub provides an efficient and scalable platform powered by modern web technologies.

# 🚀 Installation & Setup

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/Manshuchahal/EstateHub.git
cd EstateHub
```

---

## 2️⃣ Install Dependencies

### Frontend

```bash
cd client
npm install
```

### Backend

```bash
cd server
npm install
```

---

## 3️⃣ Configure Environment Variables

Create a `.env` file inside the server directory and add:

```env
PORT=5000
MONGO_URI=your_database_url
JWT_SECRET=your_secret_key
```

---

## 4️⃣ Run the Application

### Start Backend

```bash
npm start
```

### Start Frontend

```bash
npm run dev
```
